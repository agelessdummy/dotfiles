1 Introduction

This document is intended as a showcase of the mathematical abilities of eLyXer; for more information be sure to visit the main page.

1.1 Versions

There are several versions of this page:
  • A non-Unicode version of this page with midspaces.
  • A Unicode version with mathematical spaces (generated with --unicode).
  • An ISO-8859-15 version (generated with --iso885915).
  • A jsMath version that uses jsMath (generated with --jsmath).
  • A MathJax version with MathJax (generated with --mathjax).
All of them are generated from the same .lyx source file; they should help you decide which rendering options suit you best.
